Glosario de las TI (A - M)

A B C D E F G H I J K L M

A

  • Algoritmo: Secuencia de pasos para resolver un problema.
  • API: Interfaz para comunicación entre software.
  • Array: Estructura de datos indexada.
  • ASCII: Código estándar para caracteres.
  • Authentication: Verificación de identidad.

B

  • Backend: Lógica y datos del servidor.
  • Bandwidth: Capacidad de transmisión de datos.
  • Bit: Unidad mínima de información.
  • Blockchain: Cadena de bloques para registros seguros.
  • Bug: Error en software.

C

  • Cache: Memoria rápida temporal.
  • Cloud: Servicios en internet.
  • Cookie: Datos almacenados en navegador.
  • Compiler: Programa que traduce código.
  • CPU: Unidad central de procesamiento.

D

  • Database: Colección organizada de datos.
  • Debug: Proceso para encontrar errores.
  • DNS: Traducción de dominios a IP.
  • Domain: Nombre de sitio web.
  • Download: Descargar datos.

E

  • Encryption: Codificación de datos.
  • Endpoint: Punto de acceso a servicio.
  • Error: Fallo en programa.
  • Ethernet: Red cableada local.
  • Email: Correo electrónico.

F

  • Frontend: Parte visible de una aplicación.
  • Framework: Herramientas para desarrollo.
  • FTP: Protocolo para transferencia de archivos.
  • Firewall: Sistema de seguridad en red.
  • Function: Bloque de código reutilizable.

G

  • Git: Control de versiones.
  • GUI: Interfaz gráfica de usuario.
  • GPU: Procesador gráfico.
  • Gateway: Punto de acceso entre redes.
  • Gigabyte: Unidad de almacenamiento.

H

  • HTML: Lenguaje de marcado para web.
  • HTTP: Protocolo para transferencia web.
  • Hosting: Alojamiento web.
  • Hash: Función para valores únicos.
  • Hub: Dispositivo de red.

I

  • IDE: Entorno de desarrollo integrado.
  • IP: Dirección única en red.
  • IoT: Internet de las cosas.
  • Interface: Punto de interacción.
  • Iteration: Repetición en programación.

J

  • JavaScript: Lenguaje para interactividad web.
  • JSON: Formato ligero para datos.
  • JQuery: Librería de JavaScript.
  • Java: Lenguaje orientado a objetos.
  • JVM: Máquina virtual de Java.

K

  • Kernel: Núcleo del sistema operativo.
  • Keylogger: Software para registrar teclas.
  • Kubernetes: Gestión de contenedores.
  • Key: Clave para cifrado o acceso.
  • Keyword: Palabra reservada en código.

L

  • Latency: Retardo en transmisión.
  • Linux: Sistema operativo libre.
  • Load Balancer: Distribuye cargas en servidores.
  • Loop: Bucle en programación.
  • Log: Registro de eventos.

M

  • Machine Learning: Aprendizaje automático.
  • Malware: Software malicioso.
  • MySQL: Sistema gestor de bases de datos.
  • Module: Unidad de código reutilizable.
  • Memory: Memoria del sistema.
