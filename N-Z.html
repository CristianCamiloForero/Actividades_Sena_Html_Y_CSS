Glosario de las TI (N - Z)

N O P Q R S T U V W X Y Z

N

  • Node.js: Entorno de ejecución para JavaScript en el servidor.
  • Namespace: Espacio de nombres para evitar conflictos.
  • Network: Conjunto de dispositivos conectados.

O

  • Open Source: Software con código abierto.
  • OAuth: Protocolo de autorización segura.
  • Object-Oriented Programming: Paradigma basado en objetos.

P

  • PHP: Lenguaje de programación para desarrollo web.
  • Ping: Herramienta para comprobar conectividad de red.
  • Protocol: Conjunto de reglas para comunicación.

Q

  • Query: Consulta a una base de datos.
  • Queue: Estructura de datos FIFO.
  • Quantum Computing: Computación basada en qubits.

R

  • RAM: Memoria de acceso aleatorio.
  • Router: Dispositivo que dirige tráfico de red.
  • REST: Arquitectura para servicios web.

S

  • SQL: Lenguaje para gestionar bases de datos.
  • Server: Computadora que ofrece servicios en red.
  • SSL: Protocolo para comunicación segura.

T

  • TCP/IP: Protocolos fundamentales de internet.
  • Token: Elemento para autenticación o autorización.
  • Thread: Unidad de ejecución en programación.

U

  • UI: Interfaz de usuario.
  • URL: Localizador uniforme de recursos.
  • UDP: Protocolo de datagramas sin conexión.

V

  • VPN: Red privada virtual.
  • Variable: Contenedor para datos en programación.
  • Version Control: Gestión de cambios en código.

W

  • WAN: Red de área amplia.
  • Web Server: Servidor que entrega páginas web.
  • Wi-Fi: Tecnología de red inalámbrica.

X

  • XML: Lenguaje de marcado extensible.
  • XSS: Vulnerabilidad de seguridad web.
  • Xcode: Entorno de desarrollo para macOS.

Y

  • YAML: Formato de serialización de datos.
  • Yield: Palabra clave para generar valores en programación.
  • Yottabyte: Unidad de almacenamiento enorme.

Z

  • Zero-day: Vulnerabilidad desconocida y explotada.
  • Zip: Formato de compresión de archivos.
  • Zombie: Computadora comprometida y controlada remotamente.
